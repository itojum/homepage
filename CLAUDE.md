# CLAUDE.md

## コミット

作業は論理的な単位ごとに個別にコミットする。複数の独立した変更をひとつのコミットにまとめない。

## ブランチ運用

- PR は必ず `develop` ブランチへ出す。`main` への直接コミット・手動 PR は行わない
- `main` へのマージは週次デプロイ PR（GitHub Actions 自動生成）経由のみ

## テスト

このプロジェクトにテストスイートは存在しない。「テストを実行して確認」という提案は不要。

## デプロイ環境

Cloudflare Workers にデプロイされる。Node.js 組み込みモジュール（`fs`、`path` 等）は Cloudflare Workers の実行時コードでは使用不可。ビルド時・生成時の処理や設定ファイルでの利用まで一律に禁止するものではない。
